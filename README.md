## Features
- Islamic News Updates
- Daily Duas
- Authentic Hadith Collection
- Quranic Verses & Tafsir
- Bangla Islamic Articles

## Technologies Used

- CSS
- JavaScript

## Purpose
To spread authentic Islamic knowledge and news in an easy and accessible way.
